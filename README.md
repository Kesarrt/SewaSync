## 📖 Overview
**SewaSync** solves the "Coordination Crisis" in local community service. By replacing fragmented WhatsApp chats and manual logs with a unified, real-time platform, we ensure that help arrives precisely where it is needed most.

## 🛠️ Technical Stack
| Category | Technology |
| :--- | :--- |
| **Frontend** | React (Vite) + Tailwind CSS |
| **AI Layer** | Google Gemini 1.5 Flash API |
| **Backend** | Firebase Firestore (Real-time DB) |
| **Auth** | Firebase Authentication |
| **Deployment** | Vercel (Production Cloud) |
| **Messaging** | EmailJS |

## ✨ Key Features
*   **AI Insight Dashboard:** Gemini 1.5 Flash instantly categorizes raw public messages by urgency and type.
*   **Real-Time Task Sync:** Firestore listeners update volunteer dashboards in milliseconds when a mission is assigned.
*   **Proof of Completion:** Volunteers upload images/notes for Admin verification.
*   **Gamified Rewards:** A built-in Credit System recognizes volunteer contributions.
